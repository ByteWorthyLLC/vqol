# Integrations

## Runtime Integrations

- Browser IndexedDB
- Browser Notification API
- Browser service worker and PWA install surface
- Browser print dialog

## Static Assets

- `public/practice.json`
- `public/icons/*.png`
- `messages/*.json`

## External Services

None in the default app.

## Planned Integrations

- GitHub Pages public demo.
- Optional practice-owned aggregate endpoint in Phase 5.
- Optional Cloudflare Worker aggregate receiver example in Phase 5.

## Integration Rules

- No default third-party analytics.
- No default hosted API.
- No EHR integration in v1.
- Aggregate endpoints must be HTTPS and practice-owned.
