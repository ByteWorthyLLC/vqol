# Changelog

All notable changes to this project are documented here.

## Unreleased

### Added

- Static Svelte 5 PWA with hash routing.
- Local-only IndexedDB session, score, and metadata storage.
- Survey draft resume flow.
- VEINES-QOL/Sym scoring-engine shape with placeholder constants.
- Practice branding through `public/practice.json`.
- Locale files for English, Spanish, French, and German.
- Longitudinal score chart using lazy-loaded uPlot.
- Clinician print/PDF export using `window.print()`.
- Reminder scheduling helpers and in-app reminder banner.
- PWA manifest, service worker, icons, and install prompt.
- GSD planning artifacts for legal, vertical slice, white-label/i18n, and patient value phases.
- Public repo hygiene files and GitHub workflow scaffolding.

### Known Gaps

- LSHTM permission for verbatim VEINES-QOL/Sym item text and scoring constants is not yet granted.
- Public GitHub Pages demo is not deployed.
- Real-device iOS Safari PDF/PWA verification is still required before launch.
